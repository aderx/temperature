# 多点温度监测系统

## 系统说明
本系统计划采用纯前端方案制作，由于系统功能并不复杂，故采用单页应用方式开发，目前使用技术：
- 前端基础 `VUE.JS v.2`
- 后端基础 `NODE.JS`
- 数据库 `MONGODB`

为了更好的实现需求，使用了如下插件：

**前端插件**

- Axios

**后端插件**

- HTTP


为了增强系统的交互性能，使用了如下插件：

**前端插件**

- FontAwesome
- ECharts

**后端插件**

- NONE


 具体插件版本等信息在 `package.json` 

## 系统目录说明

``````
- public --> 系统主文件index index.html所在
- src --> 系统开发文件
  - assets --> 系统静态资源文件
  - components --> 系统所有组件
  - router --> 系统路由文件
  - views --> 系统所有页面
- doc --> 系统开发文档
``````
